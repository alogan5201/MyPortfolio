Pomodoro App

This is an app I built that can be used to track work session following the pomodoro technique. The app works as a timer that stores/logs work sessions upon completition. After completing one work sessions an image of a half-filled tomato will appear. Once a whole hour of work is completed, then the tomato will be completely filled. 1 hour = 1 whole tomato. Each work session is logged and categorized by Month on the archive page. Creating this app was a challenge because I decided to create it using primarily vanilla JS. Other than a bit of jQuery this app does not require any kind of front-end framework. For demonstration purposes I set work session duration to 10 seconds and the break duration to 5 seconds. For this project I used MongoDB, jQuery, nodeJS, & expressJS
